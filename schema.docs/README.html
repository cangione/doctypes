Unsupported Formats

ATA iSpec 2200 2003.1 (DTD)

Status: Not Processed

ATA iSpec 2200 uses DTD (Document Type Definition) format, not XML Schema. The xs3p stylesheet only processes W3C XML Schema (XSD) files. DTD documentation requires a different tool or manual processing.

DocBook 5.1 (RELAX NG)

Status: Not Processed

DocBook 5.1 schemas are defined in RELAX NG (RNG) format with Schematron validation rules, not XML Schema. The xs3p stylesheet is designed for W3C XML Schema (XSD) only.

Technical Details

Generation Process

  • Tool: xs3p XSLT 1.0 stylesheet
  • Processor: xsltproc (libxslt)
  • Schema Merging: xsdmerge.xsl used for multi-file schemas with xs:include and xs:redefine directives

Output Features

  • Alphabetically sorted components (elements, types, groups)
  • Complete type hierarchy including supertypes and subtypes
  • Cross-referenced links between schema components
  • Content model diagrams
  • XML source view for each component
  • Component glossary
  • Bootstrap-based responsive layout
