DITA 1.3 Schema Documentation

Complete reference documentation for DITA 1.3 top-level schema files

Overview

This documentation provides detailed schema reference information for DITA (Darwin Information Typing Architecture) version 1.3 top-level shell schemas.

Each schema file below has been processed with the xs3p documentation generator to create browsable HTML reference documentation, including:

  • Global elements and types
  • Type hierarchies and inheritance chains
  • Content models and attribute groups
  • Detailed component documentation
  • Cross-referenced links between types
